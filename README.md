# 手机维修与二手机买卖管理系统

## 项目简介
一个功能完整的手机维修与二手机买卖管理软件，支持库存管理、销售管理（零售/批发/网络）、维修管理、客户管理、报表统计等功能。

## 功能特点
- 📦 **库存管理**：商品入库、出库、库存预警
- 👥 **客户管理**：客户信息管理、分类管理
- 💰 **销售管理**：支持零售、批发、网络销售三种模式
- 🔧 **维修管理**：维修订单管理、定金收取、尾款结算
- 📊 **报表统计**：利润统计、销售报表、库存价值
- 💾 **数据备份**：一键备份数据库
